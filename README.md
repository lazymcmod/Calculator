*Qt Calculator – First Qt Project

This is a simple Calculator Application built using Qt.
It is my first project in Qt, and before this I had no prior experience with Qt framework.
Through this project, I learned how to create UI, connect signals & slots, and handle button events in Qt.

*Features

1. Basic arithmetic operations: Addition, Subtraction, Multiplication, Division
2.Clean and simple UI
3.Built using Qt Widgets
4.Beginner-friendly code

*Technologies Used

1.Qt (Qt Widgets)
2.C++
3.Qt Creator IDE

*What I Learned

1.How to create UI using Qt Designer
2.How signals and slots work
3.Handling button input and displaying results
4.Structuring a Qt Widgets project

*How to Run

1.Clone the repository:
2.git clone <your-repo-link>
3.Open the project in Qt Creator
4.Build & Run
